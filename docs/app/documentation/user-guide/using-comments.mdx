## Using Comments

### Adding a New Comment

Comments can only be added to existing documents. The Comment button is NOT available when you initially create a document.

1. Click the “Comment” button at the top of the edit form to open the Comment panel

![Comment button](/meditor/docs/images/user-guide/using-comments__comment-icon.png)

2. Click the “New Comment” button ![New comment button](/meditor/docs/images/user-guide/using-comments__new-comment-btn.png) to open the comment entry dialog.

![Comment entry dialog](https://lh5.googleusercontent.com/yDx9c4O6vu9m_xUqjmf81klofIa2gRPpyne--Zw5AqG_tKnzmH3pr5iJ-LVR2C3OOQQDRp-dAcm3DI8e0968Z_BVSeRP6fEZz5dtM2WTJfulfIUv-UPcTWd2qEl_Pul16A)

3. Enter your comment and click the “Send” button ![Send comment button](https://lh4.googleusercontent.com/Z2o5QR_6grH9sksPdEs1kIFQE_OfAVWjUVe4m0I5DrmQ9RlN59i-egOqsoAD8JUdMfhOQSh6Y6UCSL6_WsPjud_jBcxrR9ZdtmS0odhDhTbMi82112Z1nRhDvFbY0Le8Cw) to commit it, or the “Cancel” ![](https://lh3.googleusercontent.com/BuJhZcsaVPkGHRTj2wFStwDXQQm9WGt9GwdJThB-K5MEC4A88ld8CkENz6zEe7EkutxL8Yiscxd3OGKqaQq9eI4_ezE8gnyT9AxIr0mYHZTVhZNJpZrLbx_0jM5f72b2Aw)button.

### Responding to a Comment

1. Click the Comment button at the top of the edit form to open the Comment panel

![Comment button](https://lh5.googleusercontent.com/Dz_z3pMvrXEbirLAo6vS4MB3G1SquaK4JgIxbI84Ro2oZyOQmuLVcWhc1vlNQuCKtxEoV0ZjS9oC2uPBlpJBXLWubYux-ae-gSjLuaH-3_1TIUeuA9_PEBFDIrJ48NAGVg)

2. To respond to any comment, click the “Reply” button ![Reply button](https://lh5.googleusercontent.com/IpcKVbCrg6Ij7vmJ_U9iTsw2nLTu0N3IPa6HZcn0U3SyOOqHz5d3iEffQImOMfsxlbkz9ci0Ig0JAFDznjlm3fSpORcYlRe1aL9afCI6CFDik-5E8XeNt6SyjY8uiiJXxg)to open the comment reply dialog.

![Reply entry dialog](https://lh6.googleusercontent.com/AntZuxAj8lDS8Ek-Owlcuc8IBKIiMI5_lXjTRyk0c22q1f-NZEGQrpbtPw2M3Z5DxRg37QC3WupsLyT5IkdDGGypdY0b39n2occ6HnC_WZPasG6sakxwF2m9gidvMno29w)

3. Enter your response and click the “Send” button ![Send reply button](https://lh4.googleusercontent.com/Z2o5QR_6grH9sksPdEs1kIFQE_OfAVWjUVe4m0I5DrmQ9RlN59i-egOqsoAD8JUdMfhOQSh6Y6UCSL6_WsPjud_jBcxrR9ZdtmS0odhDhTbMi82112Z1nRhDvFbY0Le8Cw) to commit it, or the “Cancel” ![](https://lh3.googleusercontent.com/BuJhZcsaVPkGHRTj2wFStwDXQQm9WGt9GwdJThB-K5MEC4A88ld8CkENz6zEe7EkutxL8Yiscxd3OGKqaQq9eI4_ezE8gnyT9AxIr0mYHZTVhZNJpZrLbx_0jM5f72b2Aw)button. Responses to comments create a thread (group of related comments).

![Conversation thread](https://lh6.googleusercontent.com/Cj8YNMVVAwsmeuqqrF8-OxwZkqb303oDYtX31O8r43YUxeFyhdz_ifkAwQmsBVmYlbNhU45yjIohd-Nn-rg2E_7Yof1659iAE2Up59erjWcj7nkHlJFbURZKpdIcIP2vVw)

### Resolving a Comment

When comments and discussions have been address they need to resolved so they are no longer active.

1. Click the “Comment” button at the top of the edit form to open the Comment panel

![Comment button](https://lh6.googleusercontent.com/s2zt7k2O5GCGfQmESfsTLGuxuyVtjTWyKPx6y_FRLDeTxlulIBav8407HHZUb4Ml858R4IU-Pf6ZU8MvYKZ91GcMkpCXvEOnmhD0VEhyTASKxpzEMZl7XyfR0c8UDSGM7w)

2. To resolve a comment (comment thread) click the “Resolve” button ![Resolve comment thread button](https://lh3.googleusercontent.com/pxMplOgNEl2pyAIPvscUFdZRo9KfaxlF3L4wygqZYccBOYOfKR5iecwgxrQ56P90sCPWDWldGkySVQj2MbcOajo2Fc-Eiw5mhd6pVRzkOD1lAQr3cfYskKFPRkRDCd8YWQ) All comments with the thread are cleared from the Comment panel and the comment count in decremented by the number of comments you resolved.

![Comment button](https://lh5.googleusercontent.com/8jeq_lf6smWc2P6Nl6QZPc8HW6pgdXlqfispC6ziHB6at0SZc-mPCux7FPVDDIdeArGZ14vwXR0_KIfQQNy5Eo2eonudbtuJZ6zsXXGYygJh8nUkDQSZ8savH_lMzSPkqQ)

### Viewing Resolved Comments

By default only active comments are shown in the Comment panel. The allows the members of the workflow team to focus only on outstanding items that need to be addressed. To see the complete history of comments made open the Comment panel and click the “Show all” button ![](https://lh4.googleusercontent.com/44a8HC3MZoWzIRrj9WV_4Q7Fp-5K3pvqzg-AT17BU7DeQPQaQsgwT_c25qPMVY_TTDiD5ngqJxxy_Jlv9C0i2CCkmRHoHOzP1Evqkl45oC1sMobGSiQWQO4ZWROWgq6b_g). Resolved comments are grayed-out and can not be changed. To hide resolved comments click the “Hide resolved” button.

![Comment resolve thread](https://lh3.googleusercontent.com/p193O6mrNqK-ajc8_TfVdVVIvWjR7fymwr00B02O7xAEZdUd8MpjrSniqT0NCdqTlIDbwPXb_Vkb6bqB0YM4qN35N-FNbbVdBlK3xCyEUD_0F_PAUIjsYJJm6wZcljrxKg)
