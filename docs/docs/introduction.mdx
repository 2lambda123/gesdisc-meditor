# mEditor Documentation

This documentation covers two use cases: using mEditor through the browser and using mEditor through the API.

## User's Guide

mEditor's [User Guide](/user-guide/) can help you use mEditor through your browser. These documents introduce mEditor, the "model editor", and provides general guidance on how mEditor can be used.

## API Documentation

mEditor's [API documentation](/api) can help you use mEditor programmatically. This documentation showcases mEditor's API and lets you make requests against the live API.
